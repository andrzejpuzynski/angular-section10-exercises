<h1>Ćwiczenia z *ngFor</h1>

<h2>1. Wypisz tablicę 'frameworsk' za pomocą *ngFor i dodaj index:</h2>
<ul>
  <li *ngFor="let framework of frameworks; let i = index">
    {{ i + 1 }}. {{ framework }}
  </li>
</ul>

<br />

<h2>2. Wypisz model i markę samochodu z tablicy 'cars' za pomocą *ngFor:</h2>
<ul>
  <li *ngFor="let car of cars">{{ car.mark }} - {{ car.model }}</li>
</ul>

<br />

<h2>
  3. Wypisz model i markę samochodu z obiektu 'carsObj' za pomocą *ngFor i pipe
  keyvalue:
</h2>
<ul>
  <li *ngFor="let car of carsObj | keyvalue">
    {{ car.key }} - {{ car.value }}
  </li>
</ul>

<br />

<h2>
  4. Wypisz z tablicy 'frameworsk' tylko pierwszy i ostatni element. Dodatkowo
  należy użyć *ngIf:
</h2>
<ul>
  <ng-container
    *ngFor="let framework of frameworks; let first = first; let last = last"
  >
    <li *ngIf="first">{{ framework }}</li>
    <li *ngIf="last">{{ framework }}</li>
  </ng-container>
</ul>

<h2>
  5. Wypisz z tablicy 'frameworsk' elementy, które mają parzysty indeks.
  Dodatkowo należy użyć *ngIf (jeżeli już wiesz jak używać):
</h2>
<ul>
  <ng-container
    *ngFor="let framework of frameworks; let even = even; let i = index"
  >
    <li *ngIf="even">{{ i }} - {{ framework }}</li>
  </ng-container>
</ul>

<h2>
  6. Wypisz z w pętli obiektu 'persons' dla każdego użytkownika: Id, name, city,
  company name:
</h2>
<ul>
  <li *ngFor="let p of persons">
    {{ p.id }} - {{ p.name }} - {{ p?.address?.city }} - {{ p?.company?.name }}
  </li>
</ul>
