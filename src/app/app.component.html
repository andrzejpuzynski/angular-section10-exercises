Ćwiczenia z *ngFor

1. Wypisz tablicę 'frameworsk' za pomocą *ngFor i dodaj index:

  • {{ i }}. {{ item }}

2. Wypisz model i markę samochodu z tablicy 'cars' za pomocą *ngFor:

  • {{ item.mark }} {{ item.model }}

3. Wypisz model i markę samochodu z obiektu 'carsObj' za pomocą *ngFor i pipe keyvalue:

  • {{ item.key }} {{ item.value }}

4. Wypisz z tablicy 'frameworsk' tylko pierwszy i ostatni element. Dodatkowo należy użyć *ngIf:

  • {{ item }}
  • {{ item }}

5. Wypisz z tablicy 'frameworsk' elementy, które mają parzysty indeks. Dodatkowo należy użyć *ngIf (jeżeli już wiesz jak używać):

  • {{ item }}
    •

6. Wypisz z w pętli obiektu 'persons' dla każdego użytkownika: Id, name, city, company name:

  • {{ item.value.id }}. {{ item.value.name }} | {{ item.value.address.city }}
